# News Website

This is a simple news website built with React, powered by a news API and styled with Bootstrap.

## Features

- Displays latest news articles from various categories.
- Allows users to select a category to view news articles.
- Each news article includes a title, description, and image.
- Clicking on a news article opens the full article in a new tab.

## Technologies Used

- React.js
- Bootstrap
- [News API]-https://newsapi.org/

## Getting Started

To run this project locally, follow these steps:

1. Clone this repository to your local machine.
2. Navigate to the project directory in your terminal.
3. Install dependencies by running:
      npm install
4. Create a `.env` file in the root directory of the project and add your News API key:
REACT_APP_NEWS_API_KEY=your_api_key_here.
5. Start the development server by running:
      npm run dev
6. Open your browser and navigate to `http://localhost:3000` to view the website.

## Deployment

This project can be easily deployed to platforms like Netlify, Vercel, or GitHub Pages. Make sure to set up environment variables for your API keys in your deployment environment.
